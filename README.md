Apollo的Golang客户端。

Installation
------------

如果还没有安装Go开发环境，请参考以下文档[Getting Started](http://golang.org/doc/install.html) ，安装完成后，请执行以下命令：

```
gopm get github.com/cihub/seelog -v -g

```

*请注意*: 最好使用Go 1.8进行开发
